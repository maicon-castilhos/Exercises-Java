# ☕ Java Exercises

Repositório dedicado aos meus exercícios de Java, criado para praticar lógica de programação, estruturas de dados e orientação a objetos como parte do meu processo de aprendizado.

## 📌 Sobre

Aqui você encontrará soluções para exercícios de Java que venho resolvendo durante meus estudos, organizados por tema e nível de dificuldade. O objetivo é documentar minha evolução como desenvolvedor e servir como material de consulta e revisão.

## 🛠️ Tecnologias

- **Java** (JDK 17+)
- [IDE utilizada, ex: IntelliJ IDEA / Eclipse / VS Code]

Cada pasta contém exercícios relacionados ao tema, com o código-fonte comentado explicando o raciocínio da solução.

## 🚀 Como executar

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/java-exercises.git

# Acesse a pasta do exercício desejado
cd java-exercises/01-logica-basica

# Compile e execute
javac NomeDoArquivo.java
java NomeDoArquivo
```
